About This Project

Movie Recommender System — A dual-strategy engine built with Python & scikit-learn. It implements Content-Based Filtering (recommends similar movies based on genres, cast, crew & keywords) and Collaborative Filtering (recommends based on user behavior patterns).

This frontend connects to TMDB API to fetch live movie data, posters & details.
